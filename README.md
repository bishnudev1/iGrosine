# iGrosine

## Installation
### Prerequisites
- Node.js
- MongoDB
- Google Cloud Platform Account (for Google Maps API)
- Razorpay Account (for payment integration)

### Steps
1. **Backend Setup:**
   ```bash
   cd server
   npm install
   npm run dev
    ```
2. **Frontend Setup:**
   ```bash
   cd server
   npm install
   npm run dev
    ```
3. Get the client at
```bash
http://localhost:3000
```

4. Get the server at
```bash
http://localhost:5000/api
```
